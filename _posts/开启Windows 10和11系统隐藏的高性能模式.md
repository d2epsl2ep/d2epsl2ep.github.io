# 开启Windows 10/11系统隐藏的高性能模式

> **注意：家庭版系统不支持此功能**

Windows 10和Windows 11系统中存在隐藏的高性能模式，可以通过PowerShell命令开启。本文将详细介绍如何启用这些模式，以及遇到常见问题时的解决方案。

## 使用PowerShell命令开启

### 步骤1：以管理员身份运行PowerShell

1. 在搜索框搜索【Windows Powershell】
2. 点击【以管理员身份运行】

### 步骤2：执行创建电源计划的命令

在PowerShell窗口中粘贴并执行以下命令：

#### 节能模式：

```powershell
powercfg -duplicatescheme a1841308-3541-4fab-bc81-f71556f20b4a
```

#### 高性能模式：

```powershell
powercfg -duplicatescheme 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
```

执行成功后，您将看到类似以下输出：

```
电源方案 GUID: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx  (高性能)
```

### 步骤3：切换到高性能模式

1. 按下 `Win + I` 键打开设置
2. 点击「系统」→「电源和睡眠」→「其他电源设置」
3. 在电源选项窗口中，您应该能看到新创建的高性能电源计划
4. 选择该计划即可启用高性能模式

## 删除高性能模式

如果您需要删除已创建的电源计划：

1. 执行以下命令查看当前所有电源计划：

```powershell
powercfg /L
```

2. 从输出中找到您要删除的电源计划的GUID

3. 执行以下命令删除指定计划（将`{GUID}`替换为实际的GUID）：

```powershell
powercfg /d {GUID}
```

## 解决电源计划不可见问题（CSEnabled缺失）

有时即使命令执行成功，但在电源计划中仍然看不到高性能模式。这可能是因为注册表中的CSEnabled值缺失或出现问题。

### 解决方案

1. 以管理员身份打开命令提示符（CMD）：
   - 在搜索框搜索【命令提示符】
   - 点击【以管理员身份运行】

2. 执行以下命令：

```cmd
reg add HKLM\System\CurrentControlSet\Control\Power /v PlatformAoAcOverride /t REG_DWORD /d 0
```

3. 执行命令后，重启您的电脑

4. 重启后，再次检查电源选项，高性能模式应该已经可见

> **说明**：这个命令通过添加`PlatformAoAcOverride`注册表项来解决高级电源计划不可见的问题，这是在某些系统上CSEnabled值缺失时的有效替代方案。

## 重要警告

### 笔记本电脑休眠唤醒问题

> ⚠️ **严重警告：高性能模式可能导致笔记本电脑休眠后无法唤醒！**

在使用高性能模式时，许多笔记本用户报告遇到了休眠后无法唤醒的问题。这是因为高性能模式会修改系统的电源管理行为，可能与某些笔记本硬件的休眠/唤醒机制不兼容。

### 避免休眠问题的建议

1. **谨慎使用休眠功能**：在启用高性能模式时，尽量避免使用休眠功能，可以改用睡眠模式
2. **定期保存工作**：无论使用何种电源模式，都建议定期保存您的工作，防止意外情况导致数据丢失
3. **必要时切换回默认模式**：如果您需要使用休眠功能，建议在休眠前切换回平衡或节能电源计划

## 其他注意事项

- **仅适用于专业版及以上版本**：如前所述，Windows 10/11家庭版不支持此功能
- **电量消耗增加**：高性能模式会显著增加电量消耗，建议仅在连接电源时使用
- **发热增加**：长时间使用高性能模式可能导致设备发热增加，确保设备通风良好
- **不适合所有场景**：对于日常办公、浏览网页等轻量任务，平衡模式通常已经足够

## 总结

Windows 10/11的高性能模式可以提供更好的系统性能，但也伴随着一些潜在问题，特别是对笔记本电脑用户。在使用这些高级电源计划时，请务必注意本文中提到的警告和建议，以确保系统稳定运行。
